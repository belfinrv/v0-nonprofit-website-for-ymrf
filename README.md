# Young Minds Research Foundation (YMRF) Website

Welcome to the official repository for the Young Minds Research Foundation (YMRF), a nonprofit organization dedicated to empowering high school students through STEM research, mentorship, and scientific events.

## About YMRF

YMRF provides a platform where young ideas become research. Our mission is to bridge the gap between classroom learning and real-world scientific inquiry by pairing motivated students with experienced mentors and providing intensive training programs.

## Key Features

- **Research Mentorship Program**: Hands-on project experience with professional researchers.
- **STEM Boot Camps**: Intensive training in technical skills like Python, AI, and scientific writing.
- **Hackathons & Challenges**: Team-based competitions solving real-world problems.
- **Student Research Conference**: An annual event where students present their findings.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [Radix UI](https://www.radix-ui.com/) / [shadcn/ui](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [tw-animate-css](https://github.com/v0-app/tw-animate-css)

## Getting Started

To run the project locally:

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
